# planetaelectro-app
Aplicación Web E-commerce para curso de Desarrollo Full Stack - Talento Tech (2024)


🪐 Planeta Electro - Mobile App MVP 🚀
Aplicación móvil interactiva diseñada y desarrollada como un Producto Mínimo Viable (MVP) para el e-commerce tecnológico regional.

Este proyecto se enfocó en consolidar una interfaz de usuario fluida, liviana y orientada al rendimiento en dispositivos móviles.

🛠️ Tecnologías y Arquitectura Utilizadas
Node.js Engine Core: Arquitectura lógica estructurada para simulación de flujos de datos en tiempo real de forma asíncrona.
HTML5 & CSS3 Avanzado: Diseño semántico y estilizado modular con variables globales (:root), transiciones de alta fidelidad y animaciones fluidas de entrada.
JavaScript ES6+ (Mobile Logic): Motor interactivo dinámico encargado de la gestión del DOM, el enrutamiento de navegación interna y la automatización del flujo de consulta.
Mobile UI/UX Design: Paleta de colores optimizada en tonos violetas (Gamer/Tech) con tipografía Montserrat y Kalam, garantizando un diseño responsivo adaptativo (Mobile-First).
🚀 Características Clave de la Aplicación
Navegación Modular Nativa: Menú superior transformado en botones táctiles interactivos de alta sensibilidad (Hover Amarillo Flúor).
Zócalo Hero Compacto: Panel horizontal dinámico con tipografía estilo tiza y animación de balanceo continuo del puntero (🖱️), optimizando el espacio visual de la pantalla.
Grilla Responsiva 2x2: Sección "Nosotros" con 4 tarjetas fijas en Lila Pastel de consistencia corporativa, con iluminación táctil instantánea.
Catálogo de Productos de Vanguardia: Presentación de 6 categorías de hardware premium con etiquetas dinámicas (Nuevo, Popular, Más Vendido) integradas de manera estática para un rendimiento pluma.
Inmersión Total en Chat de WhatsApp (Modal UX): Sistema de consulta integrado a través de un Modal 100% opaco y estilizado en el color de fondo corporativo. Cuenta con un cuadro de mensaje directo y la automatización del flujo móvil: al presionar enviar, despliega la pantalla de éxito con un check ✓ y genera un retorno autónomo a la pantalla principal tras 3 segundos de cortesía, emulando el comportamiento de las aplicaciones móviles nativas más prestigiosas del mercado.
